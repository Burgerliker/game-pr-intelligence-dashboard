# Sentiment Rule v2

이 문서는 운영 감성 분류(`긍정/중립/부정`) 규칙과 사전을 공개합니다.

## 1) 라벨
- `positive` -> `긍정`
- `neutral` -> `중립`
- `negative` -> `부정`

## 2) 점수식
- 입력 텍스트: `title + description`
- 긍정 점수: `pos_score = sum(POS_DICT 매칭 가중치)`
- 부정 점수: `neg_score = sum(NEG_DICT 매칭 가중치)`
- 완화 규칙: 부정 매칭 + `개선/해결/대응/조치/보상안/재발방지/정상화/복구` 포함 시 `neg_score *= 0.75`
- 정규화 점수:
  - `denom = max(1.0, pos_score + neg_score)`
  - `raw_score = (pos_score - neg_score) / denom`
  - 긍/부정 동시 매칭 시 `raw_score *= 0.85`

## 3) 판정 임계치
- `raw_score <= -0.18` -> `부정`
- `raw_score >= 0.18` -> `긍정`
- 그 외 -> `중립`

## 4) confidence (0~1)
- 매칭 키워드 0개면 `0.15`
- 그 외:
  - `coverage = min(1.0, matched_keywords / 4.0)`
  - `separation = min(1.0, abs(raw_score) / 0.4)`
  - `confidence = 0.2 + 0.5*coverage + 0.3*separation`
  - 긍/부정 동시 매칭 시 `confidence *= 0.85`
